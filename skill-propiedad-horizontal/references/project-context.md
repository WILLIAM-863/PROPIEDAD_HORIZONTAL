# Contexto del proyecto

- Curso: Base de Datos (IS644 gr 101), UTP.
- Fecha de sustentacion: 19 al 22 de mayo de 2026.
- Requisito central: modelo ER de alta complejidad y miniaplicacion con 2 a 4 entidades.
- Restriccion tecnica: consultas SQL crudas, sin ORM.
- Restriccion operativa: dockerizado para ejecucion en dos maquinas.

## Proceso elegido para miniapp
Reserva de espacios comunes.

## Entidades MVP actuales
- unidad
- residente
- espacio_comun
- reserva_espacio

## Regla critica
No traslape de reservas activas/aprobadas en un mismo espacio.
